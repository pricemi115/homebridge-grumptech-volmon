The Volume Monitor is a dynamic platform plugin for Homebridge. This plugin will monitor the system for volumes (drives and partitions) and publish them as Homekit Battery services, where the free space on the volume is reported as the amount of charge remaining on the battery.

There are several configutation settings for the plugin. Refer to the README.md that is included with the plugin.
